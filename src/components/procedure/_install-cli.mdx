
* **Homebrew users:** Ensure that you've [added Homebrew to your PATH](https://docs.brew.sh/FAQ#my-mac-apps-dont-find-homebrew-utilities).
* **WSL users**: Ensure that you have the [Windows Subsystem for Linux installed](https://learn.microsoft.com/en-us/windows/wsl/install) before proceeding.

Install the Semgrep CLI tool and confirm the installation:

```console
# macOS users only
brew install semgrep

# macOS, Linux, or Windows Subsystem for Linux (WSL) users
python3 -m pip install semgrep

# users on Ubuntu 24.04 or later must use `pipx` instead of `pip`
python3 -m pipx install semgrep

# confirm
semgrep --version
```

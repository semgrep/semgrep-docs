Install the Semgrep CLI from PyPI using [pipx](https://pipx.pypa.io/) or [uv](https://docs.astral.sh/uv/), then confirm the installation:

```console
# install from PyPI using pipx
pipx install semgrep

# or install from PyPI using uv
uv tool install semgrep

# confirm
semgrep --version
```

Alternatively, run Semgrep from the [Docker image](https://hub.docker.com/r/semgrep/semgrep):

```console
docker run --rm semgrep/semgrep semgrep --version
```

* **WSL users**: Ensure that you have the [Windows Subsystem for Linux installed](https://learn.microsoft.com/en-us/windows/wsl/install) before proceeding.

:::info Prerequisite
- There are **no** prerequisites for users who have logged in to Semgrep App for the first time!
- For established users of the Semgrep App: To add a demo project, use a Semgrep App organization without any added project.
:::

To add the Semgrep App demo project, follow these steps:

1. Log in to the Semgrep App account and switch to an organization that does **not** include any project.
1. On the **[Projects](https://semgrep.dev/orgs/-/projects)** page, click **Explore a demo project**.
    ![Screenshot of Semgrep App Projects page with Explore a demo project button.](/img/semgrep-app-add-demo-project.png "Screenshot of Semgrep App Projects page with Explore a demo project button.")

Now you can triage findings and see vulnerabilities that Semgrep App finds in OWASP Juice Shop application.

Review [Viewing and managing findings](/semgrep-app/getting-started-with-semgrep-app/#viewing-and-managing-findings) to find out more about available actions in Semgrep App. See also the [Managing findings in Semgrep App](/semgrep-app/findings/) documentation.

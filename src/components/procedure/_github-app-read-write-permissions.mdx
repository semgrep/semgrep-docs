<summary>Expand for instructions on granting read and write access to a private GitHub Semgrep app</summary>

If you are an **existing** Semgrep user and you need to change 
your Semgrep app's permissions:

1. In GitHub, navigate to **[Settings > Developer Settings](https://github.com/settings/apps)**. You should see your Semgrep App listed in the **GitHub Apps** tab.  
1. Click **Edit** on the Semgrep app. 
1. Navigate to the **Permissions & events** section.  
1. Expand **Repository permissions** and scroll to **Contents**.  
1. Click the drop-down menu and select **Read and write**.  
1. Save these settings.  
1. Next, navigate back to the main **[GitHub Settings](https://github.com/settings/)** page. One way to do so is by clicking **Settings** in GitHub’s website breadcrumbs.  
1. In the **[Applications](https://github.com/settings/installations)** tab, locate the Semgrep app under the **Installed GitHub Apps** tab.  
1. Click **Review request** next to the *Permission updates requested* message, then choose **Accept new permissions**.

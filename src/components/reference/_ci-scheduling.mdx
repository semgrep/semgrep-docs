The following table is a summary of methods and resources to set up schedules for different CI providers.

| CI provider | Where to set schedule |
| ----------- | --------------------  |
| GitHub Actions | [`semgrep.yml` file](/semgrep-ci/sample-ci-configs#sample-github-actions-configuration-file) |
| GitLab CI/CD | [GitLab CI/CD interface](https://docs.gitlab.com/ee/ci/pipelines/schedules.html) |
| Jenkins | [Jenkins interface](https://www.jenkins.io/doc/book/pipeline/running-pipelines/#scheduling-jobs-in-jenkins) |
| Bitbucket Pipelines | [Bitbucket interface](https://support.atlassian.com/bitbucket-cloud/docs/pipeline-triggers/) |
| CircleCI | [CircleCI interface](https://circleci.com/docs/scheduled-pipelines#get-started-with-scheduled-pipelines-in-circleci) |
| Buildkite | [Buildkite interface](https://buildkite.com/docs/pipelines/scheduled-builds) |
| Azure Pipelines | [Azure interface](https://docs.microsoft.com/en-us/azure/devops/pipelines/process/scheduled-triggers?view=azure-devops&tabs=yaml)
